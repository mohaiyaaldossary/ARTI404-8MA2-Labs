# 🖼️ Image Processing Lab  
## Histogram-Based Image Enhancement

---

## 📌 Overview

This lab focuses on fundamental image enhancement techniques using histogram-based transformations.  
The purpose of the lab is to explore how pixel intensity distributions affect image contrast and how different methods can improve visual quality.

The implementation is done using Python and the `scikit-image` library, applying common intensity transformation techniques on sample images.

---

## 🔬 Topics Covered

- Intensity Rescaling (Contrast Stretching)
- Histogram Equalization
- Histogram Matching
- Image Visualization
- Histogram Analysis

---

## 🛠️ Technologies Used

- **Python**
- **NumPy**
- **Matplotlib**
- **Scikit-Image (skimage)**
