# Image Sampling, Quantization and Image Operations Lab

## 📖 About This Lab
This lab focuses on image processing using Python. 
We applied sampling and quantization techniques on grayscale images and performed image arithmetic and logical operations.

## 🖼 Tasks Performed

### 1️⃣ Image Sampling
- Reduced image resolution using a sampling factor.
- Used OpenCV resize with nearest neighbor interpolation.

### 2️⃣ Image Quantization
- Reduced grayscale levels of the image.
- Applied quantization using NumPy operations.

### 3️⃣ Image Visualization
- Displayed original, sampled, and quantized images using Matplotlib.

### 4️⃣ Image Addition
- Resized two grayscale images.
- Converted images to NumPy arrays.
- Added both images pixel-by-pixel.

### 5️⃣ Logical OR Operation
- Loaded two color images.
- Applied bitwise OR operation between them.
- Displayed the resulting image.

## 🛠 Tools Used
- Python
- NumPy
- OpenCV (cv2)
- Pillow (PIL)
- Matplotlib
- Anaconda / Jupyter

## 🎯 What I Learned
- How to downsample an image
- How to reduce grayscale levels
- How images are represented as NumPy arrays
- Performing arithmetic and logical operations on images
