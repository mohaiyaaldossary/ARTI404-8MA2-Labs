# Image Processing Lab – Thresholding & Histograms

## About
This lab covers thresholding and histogram-based image enhancement using OpenCV and scikit-image.

## Tasks
- Applied binary thresholding with multiple threshold values.
- Performed contrast stretching using percentile ranges.
- Applied histogram equalization to improve contrast.
- Used histogram matching to match color distribution between two images.

## Tools Used
Python, OpenCV, NumPy, Matplotlib, scikit-image

## Learning Outcome
Learned how to enhance image contrast and analyze image histograms.
